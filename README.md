# LLM Bayesian Probability Visualizer 🚀

This is my first AI project! It shows how **Prompts** change the way an AI "thinks" by shifting its internal math.

## 🧠 The Idea

I created this to show that AI uses **Bayesian Logic**. When you give it a prompt, you are giving it "Evidence" to help it choose the right word.

## 🛠️ How to Run This

1. Install the requirements:

   bash
   pip install -r requirements.txt

2. Run the app:
    streamlit run app.py

📊 Scenarios Included:

Medical: How guides help AI pick the right medicine.

Legal: How laws make AI language more precise.

Education: How AI explains things to kids.
